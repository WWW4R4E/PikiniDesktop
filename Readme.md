## 🧹 整理桌面的小工具





> 根据规则，将文件夹放在左边，文件排在其后；若文件夹内部有 `.ini` 文件设置了图标，则图标显示在右侧。





这个工具是用 C++ 和 Win32 API 实现的，特此纪念这段开发经历。





---





### 📌 功能亮点


- 自动整理桌面文件与文件夹


- 支持自定义文件夹图标（通过 `.ini` 配置）


- 左侧优先排序文件夹，保持整洁有序
